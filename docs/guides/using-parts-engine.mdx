---
title: Using Parts Engine
---

The `@tscircuit/parts-engine` automatically finds supplier part numbers (JLCPCB C-numbers) for your components based on their specifications and footprints.

## Installation

```bash
bun install @tscircuit/parts-engine
```

## Basic Usage

```typescript
import { jlcPartsEngine } from "@tscircuit/parts-engine"

const parts = await jlcPartsEngine.findPart({
  sourceComponent: {
    type: "source_component",
    ftype: "simple_resistor",
    resistance: "10k",
  },
  footprinterString: "0402",
})

console.log(parts.jlcpcb) // ["C25744", "C25879", "C25900"]
```

Returns up to 3 JLCPCB part numbers, prioritizing "basic parts" (included in assembly at no extra cost).

## Supported Components

| Component | Match By | Example |
|-----------|----------|---------|
| Resistors | `resistance`, package | "10k", "0402" |
| Capacitors | `capacitance`, package | "100nF", "0603" |
| Inductors | `inductance`, package | "10uH", "0805" |
| Diodes | package | "SOD-123" |
| Transistors | `transistor_type`, package | "npn", "SOT-23" |
| MOSFETs | `mosfet_mode`, `channel_type` | "enhancement", "n-channel" |
| Crystals | `frequency`, `load_capacitance` | "16MHz", "18pF" |
| Resonators | `frequency` | "16MHz" |
| Pin Headers | `pin_count`, `gender` | 8, "male" |
| Potentiometers | `max_resistance` | "10k" |

## Footprint Formats

Standard footprint strings or KiCad footprints:

```typescript
"0402", "0603", "0805"  // SMD packages
"SOT-23", "SOIC8"       // IC packages
"kicad:Resistor_SMD:R_0402_1005Metric"  // Auto-converted
```

## Example: Finding Parts for a Circuit

```typescript
import { jlcPartsEngine } from "@tscircuit/parts-engine"

const resistorParts = await jlcPartsEngine.findPart({
  sourceComponent: {
    type: "source_component",
    ftype: "simple_resistor",
    resistance: "10k",
  },
  footprinterString: "0402",
})

const capacitorParts = await jlcPartsEngine.findPart({
  sourceComponent: {
    type: "source_component",
    ftype: "simple_capacitor",
    capacitance: "100nF",
  },
  footprinterString: "0603",
})

console.log("Resistor:", resistorParts.jlcpcb[0])
console.log("Capacitor:", capacitorParts.jlcpcb[0])
```

## Caching

Results are cached automatically:

```typescript
import { cache } from "@tscircuit/parts-engine"

cache.clear() // Clear if needed
```

## Tips

- Use common packages (0402, 0603, 0805) for better part availability
- First result is usually a "basic part" (no extra assembly cost)
- If no parts found, try different package size or component value
