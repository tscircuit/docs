### Why this guide

ChatGPT’s Canvas has restrictions that make it hard to run `@tscircuit/core` or `@tscircuit/runframe` directly. Use `@tscircuit/circuit-preview` instead — a tiny bundle designed for ChatGPT that renders circuits synchronously.

### Repository convention: `HOW_TO_USE_TSCIRCUIT.md`

Add a file named `HOW_TO_USE_TSCIRCUIT.md` at your repo root so ChatGPT knows exactly how to render your circuits in the Canvas. Here’s a ready‑to‑copy template:

```
To preview circuits in the ChatGPT Canvas, load these scripts and render with JSX.

1) Add these script tags to the HTML:

<script src="https://unpkg.com/@tscircuit/circuit-preview@latest/dist/index.global.js"></script>
<script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script>

2) Render a circuit with JSX:

<script type="text/babel">
  window.tscircuit.render(
    <board width="10mm" height="10mm">
      <resistor name="R1" resistance="1k" footprint="0402" />
    </board>
  )
</script>

3) Optional: Build and inspect Circuit JSON directly:

<script type="text/babel">
  const { Circuit } = window.tscircuit
  const circuit = new Circuit()
  circuit.add(<resistor name="R1" resistance="1k" />)
  console.log(circuit.getCircuitJson())
</script>
```
