## What Are Mouse Bites?

Small perforated holes along the tab line that make it easier to snap boards apart cleanly.

## Configuration

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `mouseBites` | boolean | true | Enable/disable mouse bites |

**Mouse bite diameter** = 0.45 Ã— `tabWidth` (automatically calculated)

## When to Disable

- **V-scoring** will be used instead
- **Clean edges** required for enclosure fit
- **Automated depaneling** equipment available
