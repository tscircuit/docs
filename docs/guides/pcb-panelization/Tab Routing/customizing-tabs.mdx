## Tab Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `tabWidth` | Distance | 2mm | Tab thickness |
| `tabLength` | Distance | 5mm | Cutout length |

## Stronger Tabs (Large Boards)

<CircuitPreview
  defaultView="pcb"
  code={`
export default () => (
  <panel
    width="200mm"
    height="120mm"
    tabWidth="3mm"
    tabLength="10mm"
    boardGap="15mm"
  >
    <board width="80mm" height="50mm" routingDisabled>

    </board>

## Guidelines

| Board Size | Recommended Tab Width |
|------------|---------------------|
| Small (under 30mm) | 1-2mm |
| Medium (30-60mm) | 2-3mm |
| Large (over 60mm) | 3-4mm |
