## What Are Tabs?

Tabs are small bridges that hold boards together during manufacturing. After fabrication, you snap boards apart along these tabs.

## Generated Elements

1. **Tab Cutouts**: Rectangular cutouts on board edges
2. **Mouse Bites**: Small holes for easy breaking

## Default Configuration

| Setting | Default |
|---------|--------|
| Tab Width | 2mm |
| Tab Length | 5mm |
| Mouse Bite Diameter | 0.9mm (0.45 Ã— tabWidth) |
