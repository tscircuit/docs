---
title: tsci version
description: Print the tscircuit CLI version
---

`tsci version` prints the currently installed version of the tscircuit CLI.

## Usage

```bash
tsci version
```

You can also use the `-v` or `-V` or `--version` flags:

```bash
tsci -v
tsci --version
```

To upgrade to the latest version, see [`tsci upgrade`](./tsci-upgrade.md).
